# 세미나 초대 및 등록 시스템 설계 문서

## 1. 개요

본 문서는 세미나 초대를 위한 간단한 웹 시스템의 구조와 구현 언어 선택을
정리한 문서이다.\
목표는 다음과 같다.

-   세미나 소개 랜딩페이지 제공
-   참석 신청 기능 제공
-   신청 데이터 DB 저장
-   신청 성공/실패 페이지 제공

구성 흐름

랜딩페이지 → 신청버튼 → 정보 입력 페이지 → 성공/실패 페이지

------------------------------------------------------------------------

# 2. 전체 페이지 구조

    /
    랜딩페이지

    /register
    신청 페이지

    /api/register
    신청 API

    /success
    신청 완료 페이지

    /error
    신청 실패 페이지

------------------------------------------------------------------------

# 3. 페이지 상세 구성

## 3.1 랜딩페이지

목적 - 세미나 소개 - 참석 신청 유도

내용 예시

세미나 제목\
AI 기반 실시간 데이터 플랫폼 세미나

일시\
2026.04.15 14:00 \~ 17:00

장소\
서울 강남 컨퍼런스센터

버튼

\[참석 신청하기\]

버튼 클릭 시 `/register` 이동

------------------------------------------------------------------------

## 3.2 신청 페이지

입력 폼 예시

이름 * 회사명 * 직함 이메일 * 연락처 *

관심 분야

-   AI
-   실시간 데이터
-   IoT
-   플랫폼

개인정보 수집 동의 체크

버튼

신청하기

API 호출

POST /api/register

------------------------------------------------------------------------

## 3.3 성공 페이지

내용

신청이 정상적으로 완료되었습니다.

행사 안내 메일이 이메일로 발송됩니다.

------------------------------------------------------------------------

## 3.4 실패 페이지

내용

신청 처리 중 오류가 발생했습니다.

잠시 후 다시 시도해 주세요.

------------------------------------------------------------------------

# 5. 신청 처리 로직

→ error 페이지

# 6. 구현 언어 선택

## 옵션 1 (추천) Node.js + React

구성

Frontend React / Next.js

Backend Node.js (Express 또는 NestJS)

장점

-   개발 속도 빠름
-   Railway 배포 쉬움
-   프론트/백엔드 모두 JavaScript 사용

------------------------------------------------------------------------

구성

Frontend HTML / React

-   DB 연동 편리
